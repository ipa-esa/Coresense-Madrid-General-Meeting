Contributing via Pull Request

How to update the tables on this page:
This page is maintained as plain HTML in the project repository. To add or update your information (attendance, dietary needs):
  1. Fork or branch the repository.
  2. Edit organization.html – find the relevant <tbody> section.
  3. Copy an existing <tr>…</tr> row and fill in your details.
  4. Submit a Pull Request with a short description (e.g. "Add Jane Doe – TU Delft").
Each table has a comment block immediately above the <tbody> explaining what each column means.
